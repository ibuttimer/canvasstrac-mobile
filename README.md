# canvasstrac-mobile
Mobile client application for CanvassTrac suite

## Development Environment
The development environment:
* Nodejs v4.6.0
* npm v2.15.9
* bower v1.7.9
* gulp v3.9.1
* MongoDB v3.2.4

### Environment setup
In an appropriate folder:
* git clone https://github.com/ibuttimer/canvasstrac-mgmt.git
* cd canvasstrac-mgmt
* git submodule init
* git submodule update
* bower install
